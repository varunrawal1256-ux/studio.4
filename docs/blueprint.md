# **App Name**: Cricket Stats Hub

## Core Features:

- Data Input: Manually input or import match data, including runs, wickets, and player statistics.
- Ranking Calculation: Automatically calculate player rankings based on a weighted formula considering runs scored, wickets taken, and other relevant factors, while providing the option to customize weights and factors in order to align with diverse metrics.
- Comparative Analysis Tool: Allow users to input the characteristics for any player to compare across teams.
- Ranking Display: Display player rankings in a sortable, tabular format with key statistics.
- Data Visualization: Generate charts and graphs to visualize player performance trends.

## Style Guidelines:

- Primary color: Dark blue (#1A237E) to convey professionalism and trustworthiness.
- Background color: Very light blue (#E8EAF6).
- Accent color: Yellow-gold (#FFD700) for highlighting top performers.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look, suitable for both headlines and body text.
- Use cricket-themed icons, such as a bat, ball, wickets, and trophy.
- Use a clean and organized layout to present data effectively, using tables and charts where appropriate.
- Subtle transitions and animations when sorting or filtering data to improve user experience.
